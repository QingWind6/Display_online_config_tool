Online Configuration Generator

How to use:

  1. Select your hardware combination from the "Step 1" dropdown below.
  2. If an ePaper screen is selected, an additional "Step 2" option will appear. Please choose your driver board type.
  3. The required configuration code will be automatically generated in the "Step 3" box.
  4. Copy the entire generated code and paste it into your Custom_config.h file.

Step 2: (ePaper Only) Select Your Driver Board Type

Step 3: Copy the Generated Code
